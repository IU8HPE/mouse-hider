# GitHub Release Template

## Title

`HPE - Mouse Hider v1.0.0 (x64 + x86 Lite)`

## Body

HPE - Mouse Hider is now available with two binaries:

- `MouseHider_x64.exe` (full UI, PySide6, 64-bit Windows)
- `MouseHiderLite_x86.exe` (minimal UI, Tkinter, 32-bit Windows)

### Main features

- Auto-hide cursor after inactivity
- Instant restore on movement
- Timeout 1-300 seconds (default 10)
- Start/Stop control
- Start with Windows
- Start minimized
- Start monitoring at launch
- Auto-minimize when cursor is hidden

### Assets

- `MouseHider_x64.exe`
- `MouseHiderLite_x86.exe`
- `SHA256SUMS.txt`

### Notes

- Use x64 build for Windows 10/11 64-bit.
- Use x86 Lite build for 32-bit Windows.
