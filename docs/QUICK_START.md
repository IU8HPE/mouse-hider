# Quick Start

## 1) Pick the right executable

- `MouseHider_x64.exe` for 64-bit Windows
- `MouseHiderLite_x86.exe` for 32-bit Windows

## 2) Run the app

1. Copy the executable to a local folder.
2. Launch it.
3. Set inactivity timeout (default `10` seconds).
4. Click `Start`.

## 3) Optional startup behavior

- Enable `Start with Windows` to auto-launch at login.
- Enable `Start monitoring at launch` to begin automatically.
- Enable `Start minimized` for less intrusive startup.

## 4) Verify behavior

1. Leave the mouse still for timeout seconds.
2. Cursor should disappear.
3. Move mouse slightly.
4. Cursor should reappear immediately.

## 5) Configuration file

- File: `config.json`
- Location: next to the executable

The file is auto-created and auto-updated by the app.
